## 🚀 Quick Start

### Prerequisites

- Node.js 18+ (LTS recommended)
- npm or yarn package manager
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/MKWorldWide/MKronoSphere.git
cd MKronoSphere

# Install dependencies
npm install

# Build the project
npm run build
```

### Development

```bash
# Start development server
npm run dev

# Run tests
npm test

# Build for production
npm run build
```
